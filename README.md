# Student Attendance System

This is a full-stack web application for managing student attendance.

## [cite_start]Technology Stack [cite: 2]
- **Frontend:** React JS
- **Backend:** Node.js + Express
- **Database:** MongoDB

## [cite_start]Features [cite: 5-8]
- Add new students (Name & Roll No).
- View list of students.
- Mark attendance (Present/Absent).
- View attendance summary (Total, Present, Absent).
- Delete students.
- [cite_start]**Bonus:** Prevents duplicate Roll Numbers[cite: 26].
